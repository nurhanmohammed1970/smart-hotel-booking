# 🏨 Interactive Smart Hotel Booking System

## 📌 Project Description

This project is a terminal-based hotel booking system that simulates a real hotel front-desk experience.
It allows users to view available rooms and services, make reservations, and generate a final bill with taxes and service fees.

---

## ⚙️ Features

* View available hotel rooms and services
* Add items to customer reservation
* Display current reservation details
* Calculate final bill with taxes and gratuities
* Handle user input safely without crashing
* Interactive menu system in terminal

---

## 🧱 Programming Concepts Used

* **Abstraction (Blueprint):** Implemented using an abstract base class for all bookable items
* **Inheritance (Specialization):** Created different types like Hotel Rooms and Services
* **Encapsulation (Data Protection):** Protected sensitive data using private attributes and getter/setter methods
* **Polymorphism (Smart Behavior):** Each item calculates its cost differently

---

## ▶️ How to Run the Project

1. Open terminal
2. Navigate to the project folder
3. Run the following command:

```bash
python hotel_system.py
```
